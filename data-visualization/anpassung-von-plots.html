Mithilfe der Library Seaborn kann man einfach viele unterschiedliche Plots bilden, jedoch kann man diese nicht wirklich individuell anpassen. Zum Beispiel kann man nicht den Titel oder die Achsenbeschreibungen des Plots ändern. Um diese Funktionalität zu ermöglichen muss man die Library Matplotlib in Python verwenden, auf dieser auch Seaborn basiert.

Matplotlib bietet viele Low-Level-Funktionen um Plots in Python zu erstellen. Man hat viele Einstellungsmöglichkeiten und Kontrolle, jedoch braucht man in Vergleich zu Seaborn mehr Codezeilen um einen Plot zu erstellen. Darum ist es empfehlenswert Prototypen in Seaborn wegen der Einfachheit zu erzeugen und später diese mittels Matplotlib für eine Publikation mit mehr Details zu erstellen.

Um Matplotlib zu aktivieren, wird das folgende Import Statement benötigt.

# This line allows matplotlib plots to appear as images in the notebook
# instead of in a separate window.
%matplotlib inline

# plt is a commonly used shortcut for matplotlib
import matplotlib.pyplot as plt

Plots und Achsen anpassen

Um einen Plot in Matplotlib zu erstellen, erstellen wir eine Figur und fügen dann eine Achse zu der Figur hinzu. In der Matplotlib ist eine Achse ein einzelnes Diagramm und Plots können mehrere Achsen in einem tabellarischen Layout enthalten.

# Create a figure
f = plt.figure()

# Add an axes to the figure. The second and third arguments create a table
# with 1 row and 1 column. The first argument places the axes in the first
# cell of the table.
ax = f.add_subplot(1, 1, 1)

# Create a line plot on the axes
ax.plot([0, 1, 2, 3], [1, 3, 4, 3])

# Show the plot. This will automatically get called in a Jupyter notebook
# so we'll omit it in future cells
plt.show()

%%R
plot(c(0, 1, 2, 3), c(1, 3, 4, 3), type="l", col="dodgerblue4", , xlab="", ylab="")

Um die Darstellung anzupassen, können wir andere Methoden auf dem Achsenobjekt anwenden:

f = plt.figure()
ax = f.add_subplot(1, 1, 1)

x = np.arange(0, 10, 0.1)

# Setting the label kwarg lets us generate a legend
ax.plot(x, np.sin(x), label='sin(x)')
ax.plot(x, np.cos(x), label='cos(x)')
ax.legend()

ax.set_title('Sinusoids')
ax.set_xlabel('x')
ax.set_ylabel('y');

%%R
# Sine in R: https://rstudio-pubs-static.s3.amazonaws.com/110183_06adc5f01fc940f98fdc0822ac408de0.html
x=seq(0,10,0.1)
plot(x, sin(x), main="Sinusoids", type="l", col="dodgerblue4", xlab="x", ylab="y")
lines(x, cos(x), col="chocolate")
legend(x="bottomleft", legend=c("sin(x)", "cos(x)"), col=c("dodgerblue4", "chocolate"), lty=1)

Als Abkürzung verfügt Matplotlib mittels plt Module über mehrere Funktionen, die automatische einen Plot mit Achsen erzeugen.

# Shorthand to create figure and axes and call ax.plot
plt.plot(x, np.sin(x))

# When plt methods are called multiple times in the same cell, the
# existing figure and axes are reused.
plt.scatter(x, np.cos(x));

%%R
x=seq(0,10,0.1)
plot(x, sin(x), main="Sinusoids", type="l", col="dodgerblue4", xlab="x", ylab="y")
points(x, cos(x), col="dodgerblue4")

Das plt-Modul hat analoge Methoden zu einer Achse, so dass wir eine der obigen Darstellungen mit plt-Kurztiteln nachbilden können.

x = np.arange(0, 10, 0.1)

plt.plot(x, np.sin(x), label='sin(x)')
plt.plot(x, np.cos(x), label='cos(x)')
plt.legend()

# Shorthand for ax.set_title
plt.title('Sinusoids')
plt.xlabel('x')
plt.ylabel('y')

# Set the x and y-axis limits
plt.xlim(-1, 11)
plt.ylim(-1.2, 1.2);

Linien anpassen

Um die Eigenschaften des Plots zu ändern (z.B. die Linien in der obigen Darstellung), können wir zusätzliche Argumente in plt.plot übergeben.

plt.plot(x, np.sin(x), linestyle='--', color='purple');

%%R
x=seq(0,10,0.1)
plot(x, sin(x), type="l", lty=2, lwd=2, col="purple", xlab="", ylab="")

Die Matplotlib Dokumentation ist die einfachste Möglichkeit um welche Parameter für etwaige Methoden verfügbar sind. Eine andere Möglichkeit ist direkt das Linien-Object zu speichern.

In [1]: line, = plot([1,2,3])

Diese Linien-Objekte haben eine Vielzahl von Eigenschaften, die gesteurt werden können.

Text and LaTeX Unterstützung

In Matplotlib kann Text entweder relativ zu einem einzelnen Achsenobjekt oder zu der gesamten Figur hinzugefügt werden.

Die folgenden Befehle geben einen Text zu einer Achse hinzu:

  • set_title() - add a title
  • set_xlabel() - add an axis label to the x-axis
  • set_ylabel() - add an axis label to the y-axis
  • text() - add text at an arbitrary location
  • annotate() - add an annotation, with optional arrow

Diese Befehle geben eine Text zu ganzen Figur hinzu:

  • figtext() - add text at an arbitrary location
  • suptitle() - add a title

Jedes Textfeld kann LaTeX Ausdrücke für mathematische Formeln enthalten, solange sie in $ Zeichen eingeschlossen sind.

Das folgende Beispiel zeigt all diese Funktionen.

fig = plt.figure()
fig.suptitle('bold figure suptitle', fontsize=14, fontweight='bold')

ax = fig.add_subplot(1, 1, 1)
fig.subplots_adjust(top=0.85)
ax.set_title('axes title')

ax.set_xlabel('xlabel')
ax.set_ylabel('ylabel')

ax.text(3, 8, 'boxed italics text in data coords', style='italic',
        bbox={'facecolor':'red', 'alpha':0.5, 'pad':10})

ax.text(2, 6, 'an equation: $E=mc^2$', fontsize=15)

ax.text(3, 2, 'unicode: Institut für Festkörperphysik')

ax.text(0.95, 0.01, 'colored text in axes coords',
        verticalalignment='bottom', horizontalalignment='right',
        transform=ax.transAxes,
        color='green', fontsize=15)


ax.plot([2], [1], 'o')
ax.annotate('annotate', xy=(2, 1), xytext=(3, 4),
            arrowprops=dict(facecolor='black', shrink=0.05))

ax.axis([0, 10, 0, 10]);

# Add test to plot: https://stat.ethz.ch/R-manual/R-devel/library/graphics/html/text.html
# Use function italic: https://stackoverflow.com/a/16659354/12347616 ,
# https://stackoverflow.com/a/18965719/12347616
# Draw rect: https://www.math.ucla.edu/~anderson/rw1001/library/base/html/rect.html
# Arrows package: https://stackoverflow.com/a/47603255/12347616
# Axes title: https://stackoverflow.com/a/44618277/12347616 ,
# https://stackoverflow.com/a/7121951/12347616
plot(2, 1, pch=19, col="dodgerblue4", xlim=c(0, 10), ylim=c(0, 10),
    xlab="xlabel", ylab="ylabel", main="bold figure suptitle")
text(2, 6, pos=4, expression(paste("an equation: ", italic(E==mc^2))), cex=1.5)
text(3, 2, pos=4, "unicode: Institut für Festkörperphysik")

text(3, 8, pos=4, expression(italic("boxed italics text in data coords")), cex=0.75)
Arrows(3, 3.5, 2.2, 1.4, lwd=2, arr.type="triangle", arr.width=0.5)
text(3, 4, pos=4, "annotate")
text(4, 0.1, pos=4, col="forestgreen", "colored text in axes coords", cex=1.4)
title("axes title", adj = 0.5, line = +0.5, font.main = 1)

Anpassen eines Seaborn Plot mittels Matplotlib

Jetzt wo wir wissen wie man Plots mit Matplotlib anpassen kann, kann man die selben Funktionen verwenden um Plots mit Seaborn anzupassen, da ja Seaborn im Hintergrund Matplotlib verwendet.

In der Praxis verwendet man Seaborn, um Daten schnell zu untersuchen. Zur Feinabstimmung wird dann Matplotlib verwendet, um den Plot an die Anforderungen anzupassen.
